# Small Package API: Slugify

## Learning Objectives
- Write a small exported API
- Keep package behavior deterministic
- Handle repeated separators

## Task
Implement `Slugify`. Lowercase letters and digits are kept, whitespace and punctuation become single hyphens, and leading or trailing hyphens are removed.

Work only in the starter directory while practicing. The solution directory is a reference implementation used by the verifier.

## Expected Commands
- From this exercise: `go test ./...`
- From the repository root: `pnpm drill:check go/foundation/009-package-api-slugify`
- Verify the reference solution: `pnpm drill:check go/foundation/009-package-api-slugify --solution`
