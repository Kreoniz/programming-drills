# Structs for Small Domain Models

## Learning Objectives
- Model data with structs
- Attach behavior to domain types
- Reject invalid state at boundaries

## Task
Implement `AddItem` and `TotalCents`. Invalid quantities or prices should not be added. Total should multiply quantity by price for every item.

Work only in the starter directory while practicing. The solution directory is a reference implementation used by the verifier.

## Expected Commands
- From this exercise: `go test ./...`
- From the repository root: `pnpm drill:check go/foundation/007-structs-order-total`
- Verify the reference solution: `pnpm drill:check go/foundation/007-structs-order-total --solution`
