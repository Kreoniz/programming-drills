# Deep Readonly Transform

## Learning Objectives
- Write recursive mapped types
- Handle function leaves
- Preserve nested property types

## Task
Implement `DeepReadonly<T>`. Functions should be left unchanged; objects and arrays should become deeply readonly.

Edit files only under `starter/` while practicing. The `solution/` directory is a reference implementation used by the verifier.

## Expected Commands
- From this exercise: `tsc -p .`
- From the repository root: `node tools/drill-runner.mjs check ts-react/typescript-deep-dive/012-deep-readonly`
- Reference check: `node tools/drill-runner.mjs check ts-react/typescript-deep-dive/012-deep-readonly --solution`
