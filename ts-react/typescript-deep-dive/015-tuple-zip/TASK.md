# Tuple Zip

## Learning Objectives
- Use variadic tuple inference
- Write recursive tuple types
- Preserve literal tuple order

## Task
Implement `Zip<A, B>` so two tuples become a tuple of pairs until either input runs out.

Edit files only under `starter/` while practicing. The `solution/` directory is a reference implementation used by the verifier.

## Expected Commands
- From this exercise: `tsc -p .`
- From the repository root: `node tools/drill-runner.mjs check ts-react/typescript-deep-dive/015-tuple-zip`
- Reference check: `node tools/drill-runner.mjs check ts-react/typescript-deep-dive/015-tuple-zip --solution`
