# Rspack Federation Remote

## Learning Objectives
- Configure Rspack federation remotes
- Expose remote modules
- Share singleton React dependencies

## Task
Configure an Rspack remote named `analytics` that exposes `./Dashboard` and shares React as a singleton.

Edit files only under `starter/` while practicing. The `solution/` directory is a reference implementation used by the verifier.

## Expected Commands
- From this exercise: `node ../tests/check.mjs .`
- From the repository root: `node tools/drill-runner.mjs check ts-react/rspack/002-federation-remote`
- Reference check: `node tools/drill-runner.mjs check ts-react/rspack/002-federation-remote --solution`
