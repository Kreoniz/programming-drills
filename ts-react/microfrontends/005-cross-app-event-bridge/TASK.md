# Cross-App Event Bridge

## Learning Objectives
- Use DOM events for deployment boundaries
- Type cross-app payloads
- Return unsubscribe functions

## Task
Implement a typed event bridge over `window.dispatchEvent` and `window.addEventListener` for `cart:updated` events.

Edit files only under `starter/` while practicing. The `solution/` directory is a reference implementation used by the verifier.

## Expected Commands
- From this exercise: `node ../tests/check.mjs .`
- From the repository root: `node tools/drill-runner.mjs check ts-react/microfrontends/005-cross-app-event-bridge`
- Reference check: `node tools/drill-runner.mjs check ts-react/microfrontends/005-cross-app-event-bridge --solution`
