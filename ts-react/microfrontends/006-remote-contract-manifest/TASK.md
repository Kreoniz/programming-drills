# Remote Contract Manifest

## Learning Objectives
- Document remote deployment contracts
- Track exposed module names
- Make shared dependency versions explicit

## Task
Complete the manifest with remote name, entry URL, exposed modules, required shared versions, and a health-check endpoint.

Edit files only under `starter/` while practicing. The `solution/` directory is a reference implementation used by the verifier.

## Expected Commands
- From this exercise: `node ../tests/check.mjs .`
- From the repository root: `node tools/drill-runner.mjs check ts-react/microfrontends/006-remote-contract-manifest`
- Reference check: `node tools/drill-runner.mjs check ts-react/microfrontends/006-remote-contract-manifest --solution`
