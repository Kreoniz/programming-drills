# Package Exports Subpaths

## Learning Objectives
- Design package export maps
- Expose stable subpaths
- Hide internal modules

## Task
Complete the package export map so root, `./button`, and `./tokens` have type and import entries while internals remain unexported.

Edit files only under `starter/` while practicing. The `solution/` directory is a reference implementation used by the verifier.

## Expected Commands
- From this exercise: `node ../tests/check.mjs .`
- From the repository root: `node tools/drill-runner.mjs check ts-react/monorepo/004-package-exports-subpaths`
- Reference check: `node tools/drill-runner.mjs check ts-react/monorepo/004-package-exports-subpaths --solution`
